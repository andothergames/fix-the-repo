Heading tags

Using correctly ordered heading tags is important.

Screen readers and other assistive technologies rely on heading structure to help users navigate content. Skipping levels can confuse these tools, making it harder for visually impaired users to understand and navigate the page hierarchy.

Proper heading order helps search engines interpret content organisation and structured headings make content easier to scan.
